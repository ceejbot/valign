# valign

Vertical alignment plugin for the Atom.io editor.

Turns this:

```javascript
var
	_ = require('lodash'),
	assert = require('assert'),
	levelgraph = require('levelgraph'),
	P = require('p-promise'),
	util = require('util')
	;
```

into this:

```javascript
var
	_          = require('lodash'),
	assert     = require('assert'),
	levelgraph = require('levelgraph'),
	P          = require('p-promise'),
	util       = require('util')
	;
```

Has a keyboard shortcut.
